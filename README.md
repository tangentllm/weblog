## 📖 关于本仓库

本仓库用于系统性沉淀我在大模型领域的学习与实践成果，内容涵盖：

- 📄 **论文精读**：经典与前沿 LLM 论文的解读、核心思想梳理与实现细节分析。
- 🛠️ **技术原理**：Transformer 架构、注意力机制、MoE、长上下文、推理优化等底层原理笔记。
- 🔧 **工程实践**：模型微调（LoRA/QLoRA）、RAG、Agent 开发、量化部署等实战经验与代码示例。
- 📎 **资源汇总**：高质量课程、数据集、工具链与学习路径整理。

**目标**：建立一个结构清晰、持续更新的 LLM 知识库，既方便自己回溯，也希望能为同路人提供一点参考。
